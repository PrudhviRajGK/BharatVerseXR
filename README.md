## 🌟 Vision

**Democratizing India's Heritage Through AI-Powered Immersive Learning**

BharatVerseXR transforms traditional education by creating an immersive AI-powered metaverse where students virtually explore India's cultural heritage while learning in their native languages. Every classroom becomes a gateway to India's 5000-year legacy.

## 🎯 Problem Statement

- **Language Barriers**: 1.4 billion Indians struggle with English-only educational content
- **Cultural Disconnect**: Students learn about distant civilizations but remain disconnected from their own heritage
- **Geographic Limitations**: Rural students can't access India's cultural sites
- **Engagement Crisis**: Traditional textbook learning fails to captivate digital natives

## 🚀 Our Solution

An **AI-powered multilingual virtual learning platform** that combines:
- **Immersive XR Technology** for virtual heritage site exploration
- **Multilingual AI NPC Teacher** supporting 8+ regional languages
- **Dynamic Content Generation** using AI for personalized learning
- **Cultural Context Integration** making education meaningful and memorable

## ✨ Key Features

### 🧑‍🏫 Multilingual AI NPC Teacher
- Built with **Convai** for natural language understanding
- Supports **Hindi, Telugu, Marathi, English, Bengali, Tamil, Gujarati, Punjabi**
- Provides personalized learning experiences
- Travels with students across virtual destinations

### 🏛️ Interactive Virtual Heritage Sites
- **Charminar (Telangana)** - Architectural marvels and Nizami culture
- **Puri Jagannath Mandir (Odisha)** - Spiritual significance and traditions
- **Interactive India Map** for seamless state-to-monument navigation
- **360° Immersive Environments** with cultural accuracy

### 🎥 AI-Generated Educational Content
- **Dynamic Video Lessons** created in real-time
- **Regional Audiobooks** in native dialects
- **Interactive Storytelling** with historical context
- **Multilingual Subtitles** for accessibility

### 🧠 Smart Content Pipeline
```
Topic Input → AI Script Generation → Visual Creation → Audio Synthesis → Video Assembly
```

## 🔧 Technical Architecture

### **Frontend (Unity XR)**
- **Unity 2022.3.37f1 LTS** - Cross-platform XR development
- **C# Scripting** - Game logic and AI integration
- **XR Interaction Toolkit** - VR/AR interactions
- **NavMesh & Cinemachine** - Smooth navigation and camera control

### **AI & Content Generation**
- **Convai** - AI NPC conversation system
- **Gemini AI** - Script and content generation
- **Kokoro Pipeline** - Text-to-speech synthesis
- **MoviePy** - Automated video assembly
- **BeautifulSoup** - Educational content scraping

### **APIs & Services**
- **Unsplash API** - High-quality visual assets
- **PIL (Python Imaging Library)** - Image processing
- **Pydub & Soundfile** - Audio manipulation
- **SRT Assembly** - Multilingual subtitle generation

## 🛠️ Installation & Setup

### Prerequisites
- **Unity 2022.3.37f1** or later
- **Git** for version control
- **Python 3.8+** for AI content generation
- **Convai API Key** (free tier available)

### Quick Start

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/BharatVerseXR.git
   cd BharatVerseXR
   ```

2. **Open in Unity**
   - Launch Unity Hub
   - Click "Open" and select project folder
   - Wait for project to load and compile

3. **Configure API Keys**
   ```csharp
   // In ConvaiNPCController.cs
   public string convaiApiKey = "YOUR_CONVAI_API_KEY";
   ```

4. **Install Python Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

5. **Run the Project**
   - Press Play in Unity Editor
   - Interact with AI NPC for guided learning
